# REPOSITORIO ONDE EU COLOCO MINHAS ANOTAÇÕES SOBRE MARKDOWN

Você pode baixar e fazer uso também, é bem simples mas bem importante aprender isso, documente sempre seus codigos.

Entrando no arquivo aprendendo.md pelo github você não vai entender nada, e sim, poderia estar melhor, mais organizado, mas são anotações minhas que server para mim, mas que podem servir para você também, basta baixar o codigo e abrir em algum editor.

Imporante para documentação de codigo publico aqui no github, mostrar pra galera como se usa e as peculiaridades de tal coisa no código, além de que o uso de markdown para prompts melhora significadamente o entendimento da IA.
